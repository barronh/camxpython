====================
CAMx Python Examples
====================

Overview
========

This training live demonstrations CAMx data processing for four categories of task.
The goal is to provide basic tools and example analyses, so that attendees can adapt
examples to their specific needs. Each objective area has two examples ranging from
most simple to more complex.

Objectives:

1. Modifying Gridded Emissions.
2. Modifying Point Source Emissions.
3. Model Performance Evaluation.
4. Comparison of CAMx to Satellite data.

Go to the :doc:`auto_examples/index` page.
